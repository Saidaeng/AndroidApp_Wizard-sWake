# Android Application | Wizard's Wake

Fantasy-themed Android game featuring health points, difficulty levels, language switching, sound controls, and pause functionality.  
I was responsible for the UI and visual design of the game, including the logo, characters, game rules, and screen navigation.

---

## Project Overview

This project was developed as part of the **Mobile Application Development** course in the third year of study.

The objective was to design and develop an Android mobile application based on a topic of interest, applying the concepts learned throughout the course.

Our team redesigned and extended an existing game concept called **IHaveToFly**, transforming it into a fantasy-themed game titled **Wizard's Wake**, inspired by the Harry Potter novel and film series.  
The project focused on improving both gameplay functionality and user experience through additional features and a more cohesive visual design.

---

## Problem

The original game concept offered limited gameplay mechanics and features, which restricted the overall user experience.

It lacked flexibility in several aspects such as:

- Difficulty level adjustment
- Language settings
- Sound control
- Additional gameplay elements to improve engagement

---

## Goal

The goal of this project was to create a more complete and engaging Android game experience by introducing several new features, including:

- Health Point (HP) system  
- Multiple difficulty levels  
- Language switching  
- Sound control settings  
- Pause functionality  
- In-game guidance and rules  
- A cohesive fantasy-themed UI and visual design  

---

## My Role

- Led the **UI and visual design** of the game, including characters, interface elements, and overall game theme  
- Designed the **game logo, visual assets, and interface layout**  
- Designed the **navigation flow between game screens**  
- Collaborated with one teammate to integrate UI with gameplay features  
- Contributed to **project documentation**, including system explanations and summaries of development challenges and solutions
